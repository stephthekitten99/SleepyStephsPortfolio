MC PC

This artwork is by far one of the most off the wall concept I have every taken part in. My partner in crime Evan Dankowski was not only a really helpful source as far as the project goes, but he also created a really easy and fun work environment. I am glad I got to work with him during this project because it helped create a really awesome friendship! Additionally, it developed very differently than I thought it would. This functioning Minecraft Computer works based on a redstone build system That allows for images created within the world to showcase on the screen of the desktop showcase in the video. Through this and external links we created a computer system that takes the use to Youtube videos we had made in our Art 75 class. This creation worked as a mini gallery for creative works.
